3. 방화벽 설정 확인

#### 2024-09-24: 서버 자동시작 실패
**이슈**: 윈도우 재부팅 시 서버 미시작

**해결 과정**:
1. 시작프로그램 폴더에 바로가기 생성
2. 실행 권한 설정
3. 경로 절대 경로로 수정

#### 2024-09-23: PostgreSQL 연결 오류
**이슈**: 데이터베이스 연결 실패
**원인**: 서비스 중단 및 포트 충돌
**해결 과정**:

2. 포트 5432 충돌 해결
3. 연결 문자열 확인

### 해결 패턴 분석

#### 인프라 관련 문제
- **특징**: 서비스 중단, 연결 실패
- **주요 원인**: 설정 오류, 권한 문제
- **해결 방식**: 설정 파일 검토, 서비스 재시작

#### 코딩 관련 문제
- **특징**: 런타임 에러, 컴파일 실패
- **주요 원인**: 문법 오류, 타입 불일치
- **해결 방식**: 로그 분석, 코드 리뷰

#### 배포 관련 문제
- **특징**: 환경 차이, 의존성 문제
- **주요 원인**: 경로 설정, 환경 변수
- **해결 방식**: 환경 통일, 자동화 스크립트
